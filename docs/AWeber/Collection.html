Overview

Collection object used for groups of Resources. Collections imitate regular Hashes in most ways. You can access Resource by id via the [] method. 

   account.lists    #=> <AWeber::Collection />
   account.lists[1] #=> <AWeber::Resources::List id="1" />

Also like Hashes, you can iterate over all of it’s entries 

   account.lists.each { |id, list| puts list.name }

def each
  return unless block_given?
  @_entries.inject(1) do |count, entry|
    yield entry
    merge! next_page if is_at_max? count
    count += 1
  end
  self
end
