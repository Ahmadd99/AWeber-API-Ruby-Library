Overview

Base class for all entries. All resources have the following attributes:

  • id

  • http_etag (aliased as etag)

  • self_link (aliased as link)

  • resource_type_link (aliased as resource_type)

Resources also have attributes based on their the type of resource it is. To see a full list of attributes per resource, visit labs.aweber.com/docs/

Direct Known Subclasses

Collection, Resources::Account, Resources::Broadcast, Resources::Click, Resources::Followup, Resources::Link, Resources::List, Resources::Message, Resources::Open, Resources::Subscriber, Resources::TrackedEvent, Resources::VendorAccount

def has_many(name)
  define_method(name) do
    resource_link = instance_variable_get("@#{name}_collection_link")
    instance_var  = instance_variable_get("@#{name}")
    klass         = AWeber.get_class(name)
    
    return instance_var if instance_var
      
    collection = Collection.new(klass, get_resource(resource_link))
    instance_variable_set("@#{name}", collection)
  end
end
